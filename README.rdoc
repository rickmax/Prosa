# Propostas API

API Rest de gerenciamento de propostas Prosa.
ruby -> 2.4.1
rails -> 4.2.10

## Instalação

Após clonar o repositório, rode em seu console:

```ruby
bundle install
```

```ruby
rake db:create db:migrate db:seed
```
## Banco de Dados

MySQL -> Development
Postgresql -> Produção
